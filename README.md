# Focus Journal

## Overview

Focus Journal is an Android productivity tracking application developed using Kotlin and Jetpack Compose. It helps users improve focus, manage time effectively, and track productivity through an intuitive and user-friendly interface.

## Features

* Focus session tracking
* Productivity monitoring
* Streak tracking
* Weekly goal management
* Mood tracking
* Calendar view for activity history
* AI-powered productivity insights

## Technologies Used

* Kotlin
* Jetpack Compose
* Android Studio
* MVVM Architecture
* Room Database (if used)
* Material Design

## Purpose

The application is designed for students and professionals to build better work habits, improve concentration, and enhance productivity through data-driven tracking and analysis.

## Installation

1. Clone the repository.
2. Open the project in Android Studio.
3. Build and run the application on an Android device or emulator.
